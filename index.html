Acerca de Just Pay!

🎯 ¿Qué es Just Pay!?

Just Pay! es una aplicación web diseñada para simplificar el cálculo de gastos compartidos entre amigos, familiares o compañeros.

✨ Características:

  • 📱 Diseño responsivo para móvil y desktop
  • 🧮 Algoritmo optimizado para minimizar transferencias
  • 💱 Formato de moneda argentino (punto para miles, coma para decimales)
  • 🎨 Interfaz moderna con efectos visuales
  • 📤 Compartir resultados por WhatsApp
  • ⌨️ Atajos de teclado para acciones rápidas

🚀 Versión:

v0.3 - 2025
